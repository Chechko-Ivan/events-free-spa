# Events Free
## SPA для собирания всех бесплатных мероприятий в одном месте

https://eventsfree.by

Приложение автоматически собирает бесплатные мероприятия с различных сайтов, показывая все в одном месте.

Технологии:

Frontend:

- React/Redux
- Scss
- PWA

Backend:

- Node.js
- Express
- MongoDB
- Redis
